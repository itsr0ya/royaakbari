machi.r Prototype

Overview

machi.r is a high-fidelity, mobile-first web prototype developed for DXB111 (Assessment 1). The project focused on designing and building a personal portfolio website using semantic HTML and CSS, translating a visual concept into a functional digital interface.

The prototype was designed specifically for the iPhone 12 Pro viewport, prioritising visual hierarchy, clarity, and ease of navigation on small screens.

Concept and Visual Direction

The visual direction of machi.r is inspired by illustrated Japanese streets, particularly the work of Albert Kiefer. The design captures quiet everyday moments and focuses on mood, pacing, and atmosphere through restrained colour choices, calm typography, and generous spacing.

Target Audience

The site is intended for viewers interested in illustration, visual storytelling, and creative web design, including peers, tutors, and potential collaborators. As a personal portfolio, the goal was to present work clearly while maintaining a strong visual identity.

Tools and Technologies

The prototype was built using semantic HTML and CSS, with all code written manually in Visual Studio Code. Media queries were used to support responsive behaviour, while maintaining a mobile-first approach throughout the project.
