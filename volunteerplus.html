Volunteer+ Website

Overview

Volunteer+ is a responsive website developed for DXB111 (Assignment 2). The project explores inclusive and accessible web design by creating a platform that encourages older adults aged 65 and above in Logan City to engage in local volunteering opportunities. The website focuses on strengthening community connection while supporting users with low vision, reduced mobility, or limited digital confidence.

Target Audience

The primary audience for Volunteer+ is older adults (65+) living in Logan City who wish to remain active, socially connected, and purposeful through volunteering. Many users in this group may encounter barriers when navigating digital platforms, so the design prioritises clarity, simplicity, and confidence-building interactions.

Design Considerations

  • High colour contrast ratios to support low-vision users
  • Skip links for efficient keyboard navigation
  • Descriptive link text rather than sensory cues
  • Breadcrumb navigation to support orientation and location awareness

Tools and Technologies

The website was built using HTML and CSS, with all coding completed manually in Visual Studio Code. Responsive behaviour was achieved using media queries and CSS Grid. Generative AI tools were used only during the content-writing and image-creation stages to support clarity and visual representation. All layout, interaction, and development decisions were independently designed and implemented.
